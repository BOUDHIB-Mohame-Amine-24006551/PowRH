# PowRH
Application de gestion de ressources humaines pour du dépannage ponctuel : affichage des personnes en base avec des infos utiles (nom, statut, lieu, disponibilités, TJM) / création d'un besoin de dépannage avec filtrer des personnes / déclenchement d'un CTA parmi Tel, sms ou mail (avec pré-chargement du contenu)
